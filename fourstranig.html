Полезные теги HTML

  • html- Указывает программе просмотра страниц, что это HTML документ.
  • title- Помещает название документа в оглавление программы просмотра страниц
  • body bgcolor- Устанавливает цвет фона документа, используя значение цвета в стандарте RGB - пример: FFFF00 - желтый цвет.
  • body text- Устанавливает цвет текста документа, используя значение цвета в стандарте RGB - пример: 00ff00 - зеленый цвет.
  • pre - Обрамляет предварительно
  • h1 - Создает САМЫЙ БОЛЬШОЙ заголовок (как отдельный абзац)
  • b - Создает жирный текст
  • i - Создает наклонный текст
  • tt - Создает текст - имитирующий стиль печатной машинки.
  • cite - Используется для цитат, обычно наклонный текст.
